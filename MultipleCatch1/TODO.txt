捕获多个不同类型的异常

1. 基于Vec3D类、RangeException异常类进行修改
	1.1 将Vec3D的维数抽取出来
	1.2 将RangeException改为继承out_of_range

2. 添加zeroException， 当向量除以一个数为0时抛异常
   该异常迎继承 runtime_error

3. 重载 operator /(), 为Vec3D 类添加标量除法（向量除以一个数）
   当除数为0.0时抛异常
   IEEE 754 Rules:
   x > 0.0 : x / 0.0 = INF
   x < 0.0 : x / 0.0 = -INF
   0.0 / 0.0 = NaN (Not a Number)

   注意： 比较除数 divisior 是否为0.0 ，需要一些技巧

   divisior的绝对值减去0.0是否小于 std::numeric_limic<double> ::epsilpn()

#include <cmath>
#include <limits>
bool AreSame(double a, double b)
{
	return std::fabs(a - b) < std::numeric_limits<double>:: epsilon();
}
